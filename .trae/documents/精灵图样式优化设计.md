# 精灵图样式优化设计

## 问题分析

1. **尺寸不一致**：精灵图配置的是100px x 140px，但组件容器是80rpx x 100rpx
2. **背景图缩放**：当前使用 `background-size: cover` 可能导致图片拉伸变形
3. **样式配合问题**：边框和阴影效果与精灵图的配合需要优化
4. **缺乏视觉层次感**：当前精灵图显示效果较为平淡

## 优化方案

### 1. 调整精灵图配置

修改 `tileMapper.js` 中的精灵图配置，使其更适合组件尺寸：

```javascript
// 精灵图配置
const SPRITE_CONFIG = {
  // 每个麻将牌在精灵图中的尺寸（像素）
  tileWidth: 80, // 宽度调整为 80px，对应 80rpx
  tileHeight: 100, // 高度调整为 100px，对应 100rpx
  
  // 精灵图文件名
  sheets: {
    wan: 'sprites_wan.png',
    tiao: 'sprites_tiao.png',
    tong: 'sprites_tong.png',
    zi: 'sprites_zi.png',
    hua: 'sprites_hua.png'
  }
};
```

### 2. 优化精灵图 CSS 样式

修改 `mahjong-tile.wxss` 中的 `.tile-sprite` 样式：

```css
/* 精灵图样式 */
.tile-sprite {
  background-size: auto;

  display: flex;

  width: 100%;
  height: 100%;

  overflow: hidden;

```

### 3. 优化组件模板

修改 `mahjong-tile.wxml`，添加一个容器来更好地控制精灵图显示：

```html
<view class="mahjong-tile {{customClass}} {{disabled ? 'disabled' : ''}}" bindtap="onTap">
  <!-- 精灵图模式 -->
  <view class="sprite-container" wx:if="{{isSprite}}">
    <view class="tile-sprite" style="{{spriteStyle}}"></view>
  </view>
  
  <!-- 单张图片模式 -->
  <image wx:else class="tile-image" src="{{singleImage}}" mode="aspectFit"></image>
</view>
```

### 4. 优化组件 JavaScript 逻辑

调整 `mahjong-tile.js` 中的精灵图样式计算：

```javascript
// 更新麻将牌信息
updateTileInfo: function(tileText) {
  if (!tileText) return;
  
  const tileInfo = getTileInfo(tileText);
  if (!tileInfo) return;
  
  if (tileInfo.type === 'sprite') {
    // 精灵图模式
    const spriteStyle = {
      backgroundImage: `url(${tileInfo.sheetPath})`,
      backgroundPosition: `-${tileInfo.x}px -${tileInfo.y}px`,
      width: `${tileInfo.width}rpx`,
      height: `${tileInfo.height}rpx`,
      backgroundSize: `${tileInfo.width * 3}px ${Math.ceil(Object.keys(tileMap).filter(t => tileMap[t].sheet === tileInfo.sheet).length / 3) * tileInfo.height}px`
    };
    
    this.setData({
      tileInfo: tileInfo,
      isSprite: true,
      spriteStyle: spriteStyle,
      singleImage: ''
    });
  } else {
    // 单张图片模式
    this.setData({
      tileInfo: tileInfo,
      isSprite: false,
      spriteStyle: {},
      singleImage: tileInfo.path
    });
  }
}
```

### 5. 添加高级视觉效果

为精灵图添加悬停和点击效果：

```css
/* 精灵图悬停效果 */
.mahjong-tile:not(.disabled):hover .tile-sprite {
  transform: scale(1.05);
  box-shadow: inset 0 0 4rpx rgba(0, 0, 0, 0.5);
}

/* 精灵图点击效果 */
.mahjong-tile:not(.disabled):active .tile-sprite {
  transform: scale(0.95);
  box-shadow: inset 0 0 6rpx rgba(0, 0, 0, 0.7);
}
```

### 6. 确保响应式设计

添加响应式调整，确保在不同屏幕尺寸下都能正确显示：

```css
/* 响应式调整 */
@media screen and (max-width: 375px) {

    width: 70rpx;
    height: 90rpx;
  }
  
  .tile-sprite {
    background-size: auto !important;
  }
}

@media screen and (min-width: 768px) {
  .mahjong-tile {
    width: 100rpx;
    height: 130rpx;
  }
  
  .tile-sprite {
    background-size: auto !important;
  }
}
```

1. 麻将牌图片不再拉伸变形
2. 精灵图与组件容器尺寸匹配

2. 建议使用高质量的 PNG 格式精灵图
3. 测试不同网络环境下的加载情况
4. 考虑添加图片加载失败的容错处理

这个优化方案将确保精灵图正确显示，提升用户体验，并与组件整体样式协调一致。
