## 📖 项目简介

长沙麻将规则复杂——吃碰杠、258将、大胡叠加、扎鸟翻倍……新手常常一头雾水。**雀神养成记** 将长沙麻将的核心规则拆解为 5 个章节、10+ 种互动小游戏，由 Q 版吉祥物「辣子雀」全程引导，让学习规则像闯关一样有趣。

---

## 🎯 核心特色

| 特色 | 说明 |
|------|------|
| 🎮 **闯关式学习** | 5 个章节循序渐进，从识牌到实战，完成前一章才能解锁下一章 |
| 🌶️ **辣子雀导师** | Q 版麻雀角色全程陪伴，用对话气泡给出提示和鼓励 |
| 🧩 **多元玩法** | 找茬、连连看、反应抢答、拼图、多选推理、算术题等 10+ 种交互 |
| ⚙️ **规则自定义** | 扎鸟数量、庄家加倍、底分、是否允许小胡——完全可配 |
| 📊 **进度保存** | 自动保存学习进度、完成章节、玩家分数 |
| 💰 **麻将记账本** | 内置房间制记账工具，线下打牌也能用小程序算分 |

## 🗺️ 章节设计

### 第一章：初入江湖 🍵

> 认识麻将牌，学习起手胡

```
万子(1-9) · 条子(1-9) · 筒子(1-9) — 长沙只用108张，无字牌
```

**关卡 1 — 找茬小游戏**
- 牌桌上混入了无效牌（东南西北中发白），点击找出它们
- 限时、扣分惩罚，考验反应速度

**关卡 2 — 起手胡连连看**
- 展示一手14张牌，判断是「四喜」「六六顺」还是「板板胡」
- 答对后展示详细牌型定义

---

### 第二章：行牌修炼 🏮

> 掌握吃碰杠规则和258将

**关卡 1 — ⚡ 抢答反应堆**
- 上家/对家/下家打出一张牌，快速判断该「吃」「碰」还是「过」
- 连击加分，答错扣分

**关卡 2 — 🎯 寻找"将"军**
- 听牌状态缺一对将，从候选牌中选出正确的258将牌
- 小胡必须258作将，大胡不受限制

**关卡 3 — 🔥 258将强化训练**
- 快速判断一手小胡能否胡牌（是否有258将）
- 3道题，统计正确率

---

### 第三章：大胡风云 📚

> 认识特殊牌型，理解大胡叠加

**关卡 1 — 牌型识别**
- 展示一手14张牌，从选项中选出正确的大胡牌型
- 涵盖：碰碰胡、将将胡、清一色、七小对、缺一色、全求人

**关卡 2 — 将将胡挑战**
- 20张混合牌中选出14张258牌，组成将将胡
- 考验对258牌的敏感度

**关卡 3 — 大胡叠加**
- 一手牌可能同时满足多个大胡条件（如将将胡+碰碰胡=×12倍）
- 用标签多选的方式理解叠加规则

---

### 第四章：海底与扎鸟 🎆

> 学习海底捞月、海底炮和扎鸟翻倍

**关卡 1 — 深海探宝**
- 阅读场景描述，判断当前是「海底捞月」还是「海底炮」
- 理解庄家/闲家在海底的不同处理

**关卡 2 — 愤怒的扎鸟**
- 5 轮扎鸟计算题：
  - 哪些数字代表「中自己」？
  - 抽到N号是对谁？
  - 给定底分和鸟数，计算总分
  - 给定底分和鸟数，计算每家应付分数

---

### 第五章：实战演练 🏆

> 综合应用——清一色听牌推理

**5 道清一色听牌题目**，难度递增：
1. 标准「一四七/二五八/三六九」听牌结构
2. 七对预备 → 听 2/5/8 条
3. **九莲宝灯**：1112345678999 听 1-9 全！
4. 暗刻+顺子结构 → 听 3/6/9 万
5. 长顺子带暗刻 → 听 1/4/7 条

每道题：
- 🎴 展示13张手牌
- 🔢 1-9 选项多选
- ⏱️ 倒计时120秒
- ✅ 答对+20分，答错-5分
- 📝 详细解析

### 麻将牌精灵图系统

本项目使用 **Sprite Sheet** 技术渲染麻将牌，将 34 张麻将牌整合到一张精灵图中，通过 CSS `background-position` 定位每张牌：

- 精灵图布局：4行 × 9列
- 单牌尺寸：66px × 90px
- 支持本地/网络两种图片来源
- 自定义 `mahjong-tile` 组件封装显示逻辑

```
第1行：一万 二万 三万 四万 五万 六万 七万 八万 九万
第2行：一条 二条 三条 四条 五条 六条 七条 八条 九条
第3行：一筒 二筒 三筒 四筒 五筒 六筒 七筒 八筒 九筒
第4行：北  白  南  中  发  东  西  —  —
```

## 🚀 快速开始

### 前置要求

- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
- 微信小程序 AppID（或使用测试号）

### 运行步骤

```bash
# 1. 克隆仓库
git clone git@github.com:linxuesia/majong.git

# 2. 打开微信开发者工具
# 3. 导入项目 → 选择 majong 目录
# 4. 在 project.config.json 中填入你的 AppID（或使用测试号）
# 5. 点击「编译」运行
```

### 开发模式

在 `app.js` 中将 `DEV_MODE` 设为 `true`，即可：
- 解锁全部章节
- 显示开发工具栏（重置进度、解锁全部、清除缓存）
- 支持直接跳转到指定关卡

## 📋 规则配置

在「规则设定」页面可自定义以下参数，保存后全局生效：

| 参数 | 可选值 | 默认值 |
|------|--------|--------|
| 扎鸟数量 | 1只 / 2只 | 1只 |

| 底分 | 10 / 20 / 30 / 50 / 100 | 20 |
| 允许小胡 | 是 / 否 | 是 |

## 📝 长沙麻将规则速查

| 规则 | 说明 |
|------|------|
| **牌数** | 只用万、条、筒（108张），无字牌 |
| **258将** | 小胡必须用 2/5/8 作将牌，大胡无限制 |
| **吃碰优先级** | 碰 > 吃，只能吃上家 |
| **扎鸟** | 胡牌后翻牌，1/5/9 中自己，对应位数字中对应家 |
| **海底捞月** | 牌墙最后一张，庄家摸到后胡牌（×2倍） |
| **海底炮** | 牌墙最后一张，摸到后打出被胡 |
| **大胡叠加** | 多个大胡同时成立则倍数相乘（最高 ×12） |
