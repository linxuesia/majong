# 雀神养成记：长沙风云篇

一个卡通动画风格的长沙麻将教学小程序，通过闯关式小游戏让玩家轻松学会长沙麻将规则。

## 项目特色

- 🎮 **闯关式学习**：5个章节循序渐进教学
- 🌶️ **辣子雀导师**：Q版麻雀角色全程指导
- 🎯 **互动游戏**：找茬、连连看、解谜等多种玩法
- ⚙️ **规则自定义**：支持扎鸟数量、庄家加倍等规则配置
- 📊 **进度保存**：自动保存学习进度和成就

## 章节设计

### 第一章：初入江湖 🍵
- 识别108张牌（万、条、筒）
- 学习起手胡（板板胡、四喜、六六顺）
- 互动：找茬小游戏 + 起手胡连连看

### 第二章：行牌修炼 🏮
- 吃碰杠规则（碰>吃、只能吃上家）
- 258将规则（小胡必须258作将）
- 互动：抢答反应堆 + 寻找"将"军

### 第三章：大胡风云 📚
- 特殊牌型（碰碰胡、将将胡、清一色、七小对、全求人）
- 大胡叠加计算
- 互动：牌型拼图 + 大胡叠加算术题

### 第四章：海底与扎鸟 🎆
- 海底捞月、海底炮
- 扎鸟翻倍规则
- 互动：深海探宝 + 愤怒的扎鸟

### 第五章：实战演练 🏆
- 牌型判断与出牌策略
- 两种题型：出牌选择（discard）和胡牌判断（win）
- 5道综合应用题目
- 时间限制和分数系统
- 详细解析和规则巩固

## 技术栈

- 微信小程序原生开发
- WXML + WXSS + JavaScript
- 本地存储（游戏进度、规则配置）

## 快速开始

1. 安装微信开发者工具
2. 导入本项目
3. 在 `project.config.json` 中配置你的 AppID（或使用测试号）
4. 点击编译运行

## 规则配置

在"规则设定"页面可以自定义：
- 扎鸟数量（1只/2只）
- 庄家是否加倍
- 底分（10/20/30/50/100）
- 是否允许小胡

## 开发建议

1. **图片资源**：在 `images/` 目录添加辣子雀吉祥物和麻将牌图片
2. **音效**：建议录制长沙话配音（"碰一个"、"胡哒"、"看鸟"等）
3. **动画**：使用 WXSS 动画或 Canvas 实现牌型特效
4. **AI逻辑**：第五章需要实现简单的麻将AI算法

## 长沙麻将规则要点

- 只用万、条、筒（108张）
- 小胡必须258作将
- 碰优先于吃，只能吃上家
- 扎鸟翻倍（中1或5）
- 海底捞月/海底炮
- 特殊牌型：碰碰胡、将将胡、清一色等
