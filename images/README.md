# 麻将精灵图使用说明

## 精灵图配置

### 要求的精灵图

请将提供的完整麻将精灵图保存到本目录下，并命名为：

```
mahjong_sprite.png
```

### 精灵图尺寸和布局

- **总尺寸**：900px × 520px
- **单牌尺寸**：100px × 130px
- **行数**：4行
- **列数**：9列

### 布局说明

```
第1行（万子牌）：一万 二万 三万 四万 五万 六万 七万 八万 九万
第2行（条子牌）：一条 二条 三条 四条 五条 六条 七条 八条 九条
第3行（筒子牌）：一筒 二筒 三筒 四筒 五筒 六筒 七筒 八筒 九筒
第4行（字牌）：北 白 南 中 发 东 西
```

## 使用步骤

1. **保存精灵图**：将提供的麻将精灵图保存到本目录，命名为 `mahjong_sprite.png`
2. **检查配置**：确保 `utils/tileMapper.js` 中的配置正确
3. **测试效果**：在微信开发者工具中运行小程序，查看麻将牌是否正确显示

## 配置文件

相关配置位于 `utils/tileMapper.js`：

```javascript
// 单张精灵图配置
const SPRITE_CONFIG = {
  fileName: 'mahjong_sprite.png',
  tileWidth: 100, // 宽度（像素）
  tileHeight: 130, // 高度（像素）
  rows: 4,
  cols: 9,
  totalWidth: 900,
  totalHeight: 520
};
```

## 注意事项

1. 确保精灵图文件名和尺寸完全符合要求
2. 如果精灵图尺寸不同，需要修改 `SPRITE_CONFIG` 中的配置
3. 测试时确保网络环境正常，本地图片能够正确加载
4. 如果出现显示问题，检查控制台是否有错误信息

## 常见问题

### 1. 麻将牌显示位置不正确

**解决方案**：检查精灵图尺寸和配置是否匹配，调整 `SPRITE_CONFIG` 中的 `tileWidth` 和 `tileHeight`

### 2. 麻将牌显示拉伸变形

**解决方案**：确保精灵图中的单牌比例与组件容器比例一致

### 3. 无法加载精灵图

**解决方案**：
- 检查精灵图文件名是否正确
- 检查精灵图路径是否正确
- 检查微信开发者工具的安全设置，是否允许加载本地图片

## 测试方法

1. 在微信开发者工具中打开项目
2. 运行小程序
3. 进入游戏页面，查看麻将牌是否正确显示
4. 点击麻将牌，检查交互效果

如果一切正常，麻将牌将使用提供的精灵图正确显示，具有良好的视觉效果和交互体验。
